# Mermaid Diagram Editor

A modern, "No-Code/Low-Code" editor for Mermaid diagrams, built with Next.js 14, Tailwind CSS, and Zustand.

## Features

- **Dual-View Interface**: Real-time split-screen editing (Code + Preview).
- **No-Code Styling**: GUI controls for Fill Color, Border Color, and Stroke Width.
- **Live Preview**: Instant rendering of Mermaid diagrams.
- **Export**: Download diagrams as SVG.
- **Dark Mode**: Premium dark-themed UI.

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **State Management**: Zustand
- **Diagramming**: Mermaid.js
- **Icons**: Lucide React

## Getting Started

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Run the development server**:
   ```bash
   npm run dev
   ```

3. **Open the app**:
   Navigate to [http://localhost:3000](http://localhost:3000).

## Deployment

This project is optimized for deployment on [Vercel](https://vercel.com).

1. Push your code to a Git repository (GitHub, GitLab, Bitbucket).
2. Import the project into Vercel.
3. Vercel will automatically detect Next.js and configure the build settings.
4. Click **Deploy**.
